# Hotel Management System - CV Summary

## Project Title
**Full-Stack Hotel Management System** | React, TypeScript, Node.js, SQLite

## Project Description
Developed a production-ready hotel management system enabling real-time room booking, guest check-in/check-out, and automated billing with GST calculation. Built with modern web technologies and RESTful API architecture.

## Key Features
- **Real-time Dashboard**: Live statistics, occupancy rates, and revenue tracking
- **Room Management**: 15 rooms across 3 categories (Standard, Deluxe, Premium Suite) with availability tracking
- **Guest Management**: Complete guest lifecycle management with check-in/check-out workflows
- **Automated Billing**: Invoice generation with 12% GST calculation
- **Responsive Design**: Mobile-first UI with glassmorphism effects and smooth animations
- **Error Handling**: Comprehensive error boundaries, loading states, and user feedback

## Technical Stack

### Frontend
- **React 18** with TypeScript for type-safe component development
- **Vite** for fast development and optimized production builds
- **React Query** for efficient data fetching, caching, and state management
- **shadcn/ui** components with Tailwind CSS for modern, accessible UI
- **React Router** for client-side navigation

### Backend
- **Node.js** with Express.js for RESTful API
- **SQLite** database with automated schema initialization
- **RESTful API** with proper error handling and validation
- **CORS** configuration for secure cross-origin requests

## Technical Achievements
- Implemented React Query for automatic data synchronization and caching
- Built centralized API service layer with error handling and retry logic
- Created reusable error boundaries and loading components
- Optimized production builds with code splitting and tree shaking
- Designed responsive UI with modern CSS animations and glassmorphism effects
- Implemented input validation and error handling on both frontend and backend

## Project Highlights
- **Production-Ready**: Complete error handling, loading states, and user feedback
- **Real-time Updates**: Auto-refresh every 30 seconds for live data
- **Type Safety**: Full TypeScript implementation across frontend and backend interfaces
- **Optimized Performance**: Code splitting, lazy loading, and efficient state management
- **Developer Experience**: Comprehensive documentation, startup scripts, and environment configuration
